Завантаження JSON-файлу чату

Експорт із Telegram

  1. Відкрийте Telegram на комп’ютері (версія для настільних ПК).
  2. Перейдіть до чату, який хочете експортувати.
  3. Натисніть на три крапки у верхньому правому куті чату.
  4. Виберіть "Експортувати історію чату".
  5. У налаштуваннях виберіть формат JSON і зніміть галочки з непотрібних даних (фото, файли тощо), якщо хочете менший розмір файлу.
  6. Натисніть "Експортувати" і дочекайтеся завершення.
  7. Знайдіть файл result.json у папці завантажень і завантажте його тут.

Експорт із Discord

  1. Завантажте останню версію DiscordChatExporter (потрібен Windows або CLI для Linux/Mac).
  2. Отримайте токен бота або використайте токен вашого акаунта (докладніше тут).
  3. Відкрийте програму та виберіть канали, які хочете експортувати.
  4. Натисніть жовту кнопку в нижньому правому куті.
  5. Виберіть формат JSON.
  6. Натисніть "Export", щоб зберегти файл.
  7. Знайдіть експортований файл .json і завантажте його тут.

Експорт із WhatsApp

  1. Відкрийте WhatsApp на телефоні.
  2. Перейдіть до чату, який хочете експортувати.
  3. Натисніть на три крапки у верхньому правому куті.
  4. Виберіть "Ще", потім "Експортувати чат".
  5. Виберіть "Без медіа" або "З медіа" (останнє збільшить розмір файлу).
  6. Збережіть файл через Telegram: надішліть його собі або в інший чат.
  7. Завантажте файл із Telegram у форматі JSON (WhatsApp сам по собі не підтримує JSON, тому потрібен додатковий конвертер, наприклад, цей) і завантажте його тут.
